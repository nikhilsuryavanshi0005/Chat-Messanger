# Chat-Messenger
Socket Programming in java Project 
Real-Time Chat Application

This project is a modern, full-featured chat messenger designed for real-time communication. It allows users to connect, send and receive messages instantly, and maintain conversations in a clean and intuitive interface.

The application is built to be fast, reliable, and responsive across various devices.

Features

Real-Time Messaging: Instantly send and receive messages using WebSockets.

Multi-User Chat: Engage in one-on-one conversations.

Responsive Design: Optimized for  CUI ensuring a smooth experience on any device.

Message History: View previous messages in a conversation.
